Step-by-Step Contribution Guide

Fork the Repository

Go to the main repository page on GitHub and click the Fork button in the top right corner. This creates a personal copy of the project in your GitHub profile.

Clone Your Fork

Copy your fork's URL and clone it to your local machine using the terminal:

git clone https://github.com/YOUR_USERNAME/mashithandu-2.git

Create a Feature Branch

Navigate into the directory and create a separate branch to isolate your edits:

git checkout -b add-my-github

Add Your Info to the Database

Open data/contributors.js in your editor. Append your entry at the bottom of the array following this exact format:

contributors.js template 
  {
    username: "YOUR_GITHUB_USERNAME",
    name: "YOUR_NAME",
    quote: "YOUR_FAVORITE_QUOTE"
  }

Commit and Push

Stage the file, create a commit, and push the branch to your GitHub fork repository:

git add data/contributors.js git commit -m "feat: add my github handle to wall" git push origin add-my-github

Create a Pull Request

Go back to the original repository on GitHub. You should see a banner asking you to **Compare & pull request**. Click it, write a brief description, and submit! Once merged, you'll be on the wall.
